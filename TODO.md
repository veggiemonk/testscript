## Deferred Commands

- `replace` — in-place text replacement in files
- `chmod` — change file permissions
- `symlink` — create symbolic links
- `kill` — send signals to background processes
- `sleep` — pause script execution for a duration

## Deferred Features

- `stdin` — provide standard input to commands
- `ttyin` / `ttyout` — TTY input/output handling
- `unquote` — unquote a quoted string value
- `unix2dos` — convert line endings from Unix to DOS format
- `ContinueOnError` — option to continue script execution after a command fails
- Typo suggestions — suggest similar command names when an unknown command is used
